# color New Page

New pages render forms to make requests to create new data within our database.

### Method:
```
method="POST"
```
### Action:
```
action="/color"
```
### Inputs:
```html
<label>
        palette: <input type="text" name="color" id="color" />
    </label>
```
## Other things we want to render:

```html
<button type="submit"> Create new color</button>
```
